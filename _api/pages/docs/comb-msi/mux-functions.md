f_0(0,x_2,x_3,\\ldots,x_n) $</p>\n\n<figure class=\"image\">\n  <img src=\"/assets/images/comb/mux-shannon.svg\" alt=\"MUX based implementation of Shannon decomposition theorem.\" />\n
